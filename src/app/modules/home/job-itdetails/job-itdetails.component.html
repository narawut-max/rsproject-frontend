หน้าที่ความรับผิดชอบ
- วางแผนและทำงานให้เป็นไปตามเป้าหมายที่วางไว้
- ทำให้ลูกค้าพึงพอใจในงานบริการ
- ออกแบบระบบ นำเสนอ ติดตั้ง และให้คำปรึกษากับลูกค้า
- รับผิดชอบดูแลแก้ไขปัญหางานด้านเทคนิคทั้งหมดตามกำหนดเวลา
- ฝึกอบรมทางเทคนิคให้กับลูกค้า
- ดูแลรับผิดชอบผลิตภัณฑ์ Test , R&D ตัวใหม่ๆ
- ดำเนินการเรื่องการไป Demo สินค้ากับพนักงานขายให้กับลูกค้า
- ตรวจสอบและ Claim สินค้าตามกำหนดเวลา
- ทำรายงานส่ง KPI & Report ประจำสัปดาห์ ประจำเดือน
- ปฏิบัติงานตามที่ได้รับมอบหมายจากผู้บังคับบัญชา

คุณสมบัติเพิ่มเติม
• ปริญญาตรีด้านวิทยาการคอมพิวเตอร์ วิศวกรรม หรือสาขาอื่นๆที่เกี่ยวข้อง
• มีประสบการณ์ด้าน System Engineer อย่างน้อย 2 ปี
• สามารถสื่อสารเป็นภาษาอังกฤษได้ดีหรือหากได้ภาษจีนด้วยจะพิจารณาเป็นพิเศษ
• มีประสบการณ์ด้านการตืดตั้ง Windows Server
• มีความมุ่งมั่นในการทำงานให้ประสบความสำเร็จ
• มีความอยากเรียนรู้ เติบโตและพัฒนาตนเองตลอดเวลา
• มีทัศนคติที่ดี มีความกระตือรือร้น และรักงานด้านการให้บริการ
• มีทักษะในการสื่อสารที่มีประสิทธิภาพทั้งการนำเสนอและการเขียน
• สามารถทำงานเป็นทีมได้ดี
• มีความสามารถด้านภาษาอังกฤษสามารถเข้าใจ Manual หรือค้นคว้าข้อมูลที่เป็นภาษาอังกฤษได้
• มีรถยนต์เป็นของตนเองและมีใบอนุญาตขับขี่ จะได้รับการพิจารณาเป็นพิเศษ
